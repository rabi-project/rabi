# Rabi brand assets

The mark is a **Rabi oscillation**: the population transfer curve
P(t) = sin²(Ωt/2) driven between the |0⟩ and |1⟩ levels — the physics the
project is named for. All assets are original SVGs generated by
`docs/brand/` tooling; no third-party artwork.

| file | use |
|---|---|
| `rabi-logo-light.svg` | full logo on light backgrounds |
| `rabi-logo-dark.svg` | full logo on dark backgrounds |
| `rabi-mark.svg` | mark only (favicons, avatars on light) |
| `rabi-mark-tile.svg` | mark on tile (org avatar, social) |

Palette: ink `#1a1a2e` · paper `#f2f3f7` · teal `#2a9d8f` (dark-mode teal
`#3ec9b8`) · muted `#8a8fa3`.
